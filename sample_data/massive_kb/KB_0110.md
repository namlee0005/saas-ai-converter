# KB-0110: How to configure Manufacturing Module

**Module:** Manufacturing
**Tags:** beginner, beginner
**Last Updated:** 2026-04-09

## 1. Overview
This document covers the details regarding how to configure manufacturing module. It is intended for users facing challenges or needing a complete walk-through of the Manufacturing system.

## 2. Step-by-Step Guide
1. Navigate to the Manufacturing dashboard.
2. Select the settings gear icon in the top right.
3. Click on 'How Settings'.
4. Ensure your role has 'beginner' level permissions.
5. Save changes and restart the daemon if necessary.

## 3. Common Error Codes
- **ERR-MAN-001:** Invalid configuration parameters. Resolution: Check line 42 of your config.
- **ERR-MAN-002:** Permission denied. Resolution: Escalate to an administrator.
- **ERR-MAN-003:** API rate limit exceeded. Resolution: Upgrade your SaaS tier.

## 4. API Reference
To automate this via REST API, hit the endpoint:
`POST /api/v2/manufacturing/config`
Payload:
```json
{
    "action": "how",
    "force": true,
    "notify_admin": false
}
```
