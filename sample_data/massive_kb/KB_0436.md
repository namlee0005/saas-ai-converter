# KB-0436: Integration guide for Analytics Module

**Module:** Analytics
**Tags:** admin-only, beginner
**Last Updated:** 2026-04-09

## 1. Overview
This document covers the details regarding integration guide for analytics module. It is intended for users facing challenges or needing a complete walk-through of the Analytics system.

## 2. Step-by-Step Guide
1. Navigate to the Analytics dashboard.
2. Select the settings gear icon in the top right.
3. Click on 'Integration Settings'.
4. Ensure your role has 'admin-only' level permissions.
5. Save changes and restart the daemon if necessary.

## 3. Common Error Codes
- **ERR-ANA-001:** Invalid configuration parameters. Resolution: Check line 42 of your config.
- **ERR-ANA-002:** Permission denied. Resolution: Escalate to an administrator.
- **ERR-ANA-003:** API rate limit exceeded. Resolution: Upgrade your SaaS tier.

## 4. API Reference
To automate this via REST API, hit the endpoint:
`POST /api/v2/analytics/config`
Payload:
```json
{
    "action": "integration",
    "force": true,
    "notify_admin": false
}
```
