# KB-0239: Security settings in Finance Module

**Module:** Finance
**Tags:** billing, beginner
**Last Updated:** 2026-04-09

## 1. Overview
This document covers the details regarding security settings in finance module. It is intended for users facing challenges or needing a complete walk-through of the Finance system.

## 2. Step-by-Step Guide
1. Navigate to the Finance dashboard.
2. Select the settings gear icon in the top right.
3. Click on 'Security Settings'.
4. Ensure your role has 'billing' level permissions.
5. Save changes and restart the daemon if necessary.

## 3. Common Error Codes
- **ERR-FIN-001:** Invalid configuration parameters. Resolution: Check line 42 of your config.
- **ERR-FIN-002:** Permission denied. Resolution: Escalate to an administrator.
- **ERR-FIN-003:** API rate limit exceeded. Resolution: Upgrade your SaaS tier.

## 4. API Reference
To automate this via REST API, hit the endpoint:
`POST /api/v2/finance/config`
Payload:
```json
{
    "action": "security",
    "force": true,
    "notify_admin": false
}
```
