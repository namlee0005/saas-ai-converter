# KB-0169: Integration guide for Payroll Module

**Module:** Payroll
**Tags:** admin-only, api
**Last Updated:** 2026-04-09

## 1. Overview
This document covers the details regarding integration guide for payroll module. It is intended for users facing challenges or needing a complete walk-through of the Payroll system.

## 2. Step-by-Step Guide
1. Navigate to the Payroll dashboard.
2. Select the settings gear icon in the top right.
3. Click on 'Integration Settings'.
4. Ensure your role has 'admin-only' level permissions.
5. Save changes and restart the daemon if necessary.

## 3. Common Error Codes
- **ERR-PAY-001:** Invalid configuration parameters. Resolution: Check line 42 of your config.
- **ERR-PAY-002:** Permission denied. Resolution: Escalate to an administrator.
- **ERR-PAY-003:** API rate limit exceeded. Resolution: Upgrade your SaaS tier.

## 4. API Reference
To automate this via REST API, hit the endpoint:
`POST /api/v2/payroll/config`
Payload:
```json
{
    "action": "integration",
    "force": true,
    "notify_admin": false
}
```
